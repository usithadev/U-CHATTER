# U-CHATTER
This is a very simple chatbot for telegram messanger.     

You can use this bot at https://t.me/u_chatter_chatbot.

- Language: Typescript
- License: MIT  

Run on your local device:  

```bash
git clone https://github.com/usithadev/U-CHATTER.git  
cd U-CHATTER  
npm install  
npm run build  
npm start  
```
